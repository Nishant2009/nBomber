### [+] Installation
Run these commands to Install Bomber.

```shell script
git clone https://github.com/Nishant2009/Bomber 
cd Bomber
pip install bomber-1.0-py3-none-any.whl --force-reinstall
cd .. 
rm -rf Bomber
```

## [+] USAGE

To use the bomber type the following commands:
```shell script

```
